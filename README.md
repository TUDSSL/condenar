# TURNER Games Console

This is the official public repository for the TURNER battery-free gaming console.

## Folder structure
- Data: Contains the data used in the user studies along with the scripts necessary to analyze them
- Hardware: Contains the source files for the hardware used in this project. Schematics, PCBs, and 3D models.
- Software: All the different pieces of software are here (Console firmware, apps, logging software, PC programs,...)

## Preliminary studies
The software and hardware used for the preliminary studies, along with the data obtained from them can be found in the [PreliminaryStudy](https://github.com/TUDSSL/condenar/tree/PreliminaryStudy) branch.
