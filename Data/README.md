# Data Relating to Console Evaluation and User Study

This part of the repository includes all data relating to the evaluation and study of the TURNER system. The files are arranged as such: 

- ConsoleBenchmarks: In this folder you will find benchmarks used in the initial [thesis work](https://repository.tudelft.nl/file/File_07b732e0-c004-4465-9b0d-f7f7da38e547?preview=1).
  
- Documentation: Useful commands and other related console documentation.
  
- MainStudy: Everything related to the main user study (from raw logs to results)
