# Folder Structure

This folder contains the hardware design files for the project, including component lists, enclosure designs, and printed circuit board layouts required to manufacture and assemble the system. Here you will find: 

- [BOM](https://github.com/TUDSSL/condenar/tree/master/Hardware/BOM): Bills of Materials listing all components required to assemble the hardware.

- [Enclosure](https://github.com/TUDSSL/condenar/tree/master/Hardware/Enclosure): Mechanical design files and documentation for the console casing.

- [PCBs](https://github.com/TUDSSL/condenar/tree/master/Hardware/PCBs): PCB design files for all hardware modules of the TURNER system.
